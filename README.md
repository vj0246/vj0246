## 🚀 Featured Projects

### [Multi-Horizon Transformer — Equity Direction Forecasting](https://github.com/vj0246/Multi-Horizon-Transformer-for-Systematic-Equity-Direction-Forecasting)
A custom Transformer encoder that predicts **Nifty 50 direction across 20 horizons simultaneously** (next-day → 20-day) from 18+ years of OHLCV. Sinusoidal positional encoding, 60-day sliding windows, strict temporal train/val/test (no look-ahead leakage). The twist: raw logits are treated as **alpha signals** and evaluated with quant metrics — Spearman IC, sign/quantile Sharpe, decile return attribution — alongside a FinBERT news-sentiment track and an XGBoost baseline.
`TensorFlow` `Keras` `FinBERT` `XGBoost` `quant backtesting`
